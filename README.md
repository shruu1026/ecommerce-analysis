# E-commerce Data Analysis Project

## 📊 Overview
This project analyzes e-commerce order data to understand business performance and trends.

## 🔧 Tools Used
- Python (Pandas)
- VS Code
- GitHub

## 📈 Key Insights
- Around 97% of orders are successfully delivered
- Orders increased significantly from 2017 to 2018
- Highest sales observed in August and mid-year months

## 📁 Files
- analysis.py → Python code for data analysis
- olist_orders_dataset.csv → dataset used

## 🚀 What I Learned
- Data cleaning using pandas
- Handling missing values
- Extracting insights from real-world dataset
